== Cilium HelmChartConfig kpt Package

@codebase This package carries the core `+rke2-cilium+` HelmChartConfig
that bootstraps the cluster CNI. It mirrors the previous NoCloud
manifest so the values now live in git and are rendered/applied through
`+rke2-kpt-deploy+`.

=== Usage

[arabic]
. `+kpt pkg get+` this folder into the downstream cluster repository
under `+kpt/system/cilium+`.
. Adjust `+setters.yaml+` (or set values via
`+kpt fn eval -- -- cluster-name=... cluster-id=...+`) so the Helm chart
receives the right metadata for your cluster.
. Commit the rendered package. `+rke2-kpt-package-sync+` will mirror it
onto the control-plane host and `+rke2-cilium-config.service+` applies
it automatically during bootstrap.
